# 🌱 ClimateAct — Daily Carbon Footprint Tracker

ClimateAct is an AI-powered web app that helps users track and reduce their daily carbon footprint. Enter your daily activities across transportation, energy use, and food — and get a personalized emissions report with actionable recommendations generated by Google Gemini.

---

## Features

- **Carbon Footprint Calculator** — Estimates daily CO₂ emissions across three categories: transportation, energy, and food
- **AI-Powered Recommendations** — Uses Google Gemini (via LangGraph) to generate tailored advice based on your impact level
- **Impact Classification** — Automatically classifies your footprint as Low, Moderate, or High, and routes to the appropriate AI agent
- **Session Logging** — Saves each session as a JSON report in a local `carbon_logs/` directory
- **Reports Dashboard** — Browse all past sessions sorted by date
- **Climate Tips Page** — Curated tips across transportation, energy, food, and lifestyle
- **REST API** — A `/api/calculate` endpoint for programmatic access

## Getting Started

### Prerequisites

- Python 3.9+
- A [Google Gemini API key](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/climateact.git
   cd climateact
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure your API key**

   Open `.env` and replace the placeholder with your actual key:
   ```env
   GEMINI_API_KEY="your-actual-gemini-api-key"
   FLASK_ENV=development
   FLASK_DEBUG=True
   ```

4. **Run the app**
   ```bash
   python connect.py
   ```

5. Open your browser and visit `http://localhost:5000`

## How It Works

The core of ClimateAct is a **LangGraph state machine** that processes user data through a pipeline of agents:

```
User Input
    │
    ▼
Data Validator       ← Sanitizes and normalizes all inputs
    │
    ▼
Emissions Calculator ← Applies CO₂ emission factors per activity
    │
    ▼
Impact Classifier    ← Labels impact as Low / Moderate / High
    │
    ▼
  ┌─────────────────────────┐
  │   Conditional Router    │
  └──────┬──────┬───────────┘
         │      │         │
       Low   Moderate   High
    Impact   Impact    Impact
    Agent    Agent     Agent
         │      │         │
         └──────┴─────────┘
                │
                ▼
        Session Logger       ← Saves JSON report to carbon_logs/
                │
                ▼
       Final Summary Agent   ← Generates motivational closing message
```

### Emission Factors Used

| Activity | CO₂ Factor |
|---|---|
| Car travel | 0.21 kg / km |
| Public transport | 0.05 kg / km |
| Flight | 0.25 kg / km |
| Air conditioning | 1.50 kg / hour |
| Heater | 2.00 kg / hour |
| Electricity | 0.50 kg / kWh |
| Water heating | 1.20 kg / hour |
| Meat meal | 3.30 kg / meal |
| Vegetarian meal | 0.90 kg / meal |

## API Usage

ClimateAct exposes a REST endpoint for direct integration:

**`POST /api/calculate`**

```bash
curl -X POST http://localhost:5000/api/calculate \
  -H "Content-Type: application/json" \
  -d '{
    "car_km": 20,
    "public_transport_km": 5,
    "flight_km": 0,
    "ac_hours": 4,
    "heater_hours": 0,
    "meat_meals": 1,
    "vegetarian_meals": 2,
    "electricity_kwh": 3,
    "water_heating_hours": 1,
    "location": "Mumbai",
    "season": "summer"
  }'
```

The response includes `total_emissions`, `emissions_breakdown`, `impact_level`, `recommendations`, and `final_summary`.

## CLI Mode

You can also run the tracker directly in your terminal without the web interface:

```bash
python app.py
```

This launches an interactive prompt to enter your daily activities and prints the full report to the console.

## Impact Levels

| Level | Daily CO₂ | Message |
|---|---|---|
| 🟢 Low Impact | < 5 kg | Below average — keep it up! |
| 🟡 Moderate Impact | 5 – 15 kg | Room for improvement |
| 🔴 High Impact | > 15 kg | Consider making changes |

**For reference:** The global daily average is ~12 kg CO₂/person, and the Paris Agreement target works out to ~6.3 kg/day.
