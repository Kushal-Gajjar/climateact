# рҹҢұ ClimateAct вҖ” Daily Carbon Footprint Tracker

ClimateAct is an AI-powered web app that helps users track and reduce their daily carbon footprint. Enter your daily activities across transportation, energy use, and food вҖ” and get a personalized emissions report with actionable recommendations generated by Google Gemini.

---

## Features

- **Carbon Footprint Calculator** вҖ” Estimates daily COвӮӮ emissions across three categories: transportation, energy, and food
- **AI-Powered Recommendations** вҖ” Uses Google Gemini (via LangGraph) to generate tailored advice based on your impact level
- **Impact Classification** вҖ” Automatically classifies your footprint as Low, Moderate, or High, and routes to the appropriate AI agent
- **Session Logging** вҖ” Saves each session as a JSON report in a local `carbon_logs/` directory
- **Reports Dashboard** вҖ” Browse all past sessions sorted by date
- **Climate Tips Page** вҖ” Curated tips across transportation, energy, food, and lifestyle
- **REST API** вҖ” A `/api/calculate` endpoint for programmatic access

## Getting Started

### Prerequisites

- Python 3.9+
- A [Google Gemini API key](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/climateact.git
   cd climateact
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure your API key**

   Open `.env` and replace the placeholder with your actual key:
   ```env
   GEMINI_API_KEY="your-actual-gemini-api-key"
   FLASK_ENV=development
   FLASK_DEBUG=True
   ```

4. **Run the app**
   ```bash
   python connect.py
   ```

5. Open your browser and visit `http://localhost:5000`

## API Usage

ClimateAct exposes a REST endpoint for direct integration:

**`POST /api/calculate`**

```bash
curl -X POST http://localhost:5000/api/calculate \
  -H "Content-Type: application/json" \
  -d '{
    "car_km": 20,
    "public_transport_km": 5,
    "flight_km": 0,
    "ac_hours": 4,
    "heater_hours": 0,
    "meat_meals": 1,
    "vegetarian_meals": 2,
    "electricity_kwh": 3,
    "water_heating_hours": 1,
    "location": "Mumbai",
    "season": "summer"
  }'
```

The response includes `total_emissions`, `emissions_breakdown`, `impact_level`, `recommendations`, and `final_summary`.
